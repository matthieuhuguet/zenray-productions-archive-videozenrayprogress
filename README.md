# zenray-productions-archive-videozenrayprogress

Archive **ZenRay Create** du dossier de projet « VideoZenRayProgress ».

Ce dépôt fait partie de **ZenRay Progress** — un espace de travail façon Finder/Notion qui stocke ses médias
sur GitHub et les sert via CDN. Voir le projet : https://zenray-progress.vercel.app

## Ce que contient ce dépôt

Des **morceaux binaires de 19 Mo** (`aa/bb/<id>-NNN`), nommés par hachage du chemin d'origine. Il n'y a
volontairement **aucun fichier directement consultable** ici : un morceau seul n'est pas un fichier utilisable.

Le **manifeste** qui décrit chaque fichier (nom réel, arborescence, ordre des morceaux, empreinte sha256) est
**privé** et n'est pas dans ce dépôt. La reconstitution bit à bit se fait uniquement depuis l'application
ZenRay Create, qui réassemble les morceaux dans l'ordre puis vérifie l'empreinte.

## Version de clonage

Cette archive est figée sur le commit courant de ce dépôt (chaque manifeste privé référence ce commit). Une
nouvelle archive du même dossier produit un nouveau commit ; les anciens morceaux ne sont plus référencés.
